## Configuration 
Pour exécuter ce projet, installez-le localement en utilisant npm : 

$ cd ../Test_Technique_Edacy 
$ npm install 
$ ng serve 
$ Navigate to `http://localhost:4200/`

## Unit tests

Pour lancer les tests :

$ ng test
